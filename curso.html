HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y dar formato al contenido en las páginas web. Esencialmente, HTML define cómo se presenta el contenido en un navegador web, utilizando etiquetas y elementos para indicar el tipo de contenido (como texto, imágenes, enlaces, etc.). Aunque HTML no es un lenguaje de programación completo, es fundamental para el desarrollo web debido a su simplicidad y accesibilidad, que nos permite a los desarrolladores y diseñadores web crear páginas web interactivas y atractivas, organizando el contenido de manera lógica y estéticamente agradable

Extructura

.DOCTYPE html>: Declara que el documento es un documento HTML5.
.html>: Es la raíz de todos los documentos HTML.
.head>: Contiene metainformación sobre el documento, como el título.
.meta charset="UTF-8">: Permite que la página web muestre correctamente caracteres de diferentes idiomas, como acentos, letras no latinas, símbolos y emojis.
.title>: Define el título de la página, visible en la barra de título del navegador.
.body>: Contiene el contenido principal de la página web.

El header>: Define la sección de encabezado de un documento o una sección específica. Contiene elementos de encabezado y navegación.
.main>: Define el contenido principal y único de un documento. Solo debe haber un main> por página.
.section>: Agrupa contenido temáticamente relacionado. Se utiliza para dividir el contenido en bloques manejables.
.footer>: Define el pie de página de un documento o una sección específica. Contiene información de contacto, derechos de autor, y enlaces importantes.

Con la etiquetas header, main, section, footer no son tan indespensables, pero son buenas practicas y nos ayudan a organizar nuestra informacion de manera que el código se más limpio y organizado que sea más fácil de leer y mantener. Los desarrolladores pueden entender rápidamente la estructura del documento.
